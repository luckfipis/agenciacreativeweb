# 💈 Projeto Barbearia

Landing page para barbearia com design moderno, desenvolvida com HTML, CSS e **Bootstrap**.

---

## 🚀 Funcionalidades

- ✅ Layout responsivo para desktop e mobile
- ✅ Seção de serviços e preços
- ✅ Seção de agendamento
- ✅ Informações de contato e localização
- ✅ Design clean com identidade visual de barbearia

## 📦 Como rodar o projeto

1. Clone o repositório:
```bash
git clone https://github.com/luckfipis/projetobarbearia.git
```

2. Acesse a pasta do projeto:
```bash
cd projetobarbearia
```

3. Abra o arquivo `index.html` no navegador — sem necessidade de instalar nada!
